How to Optimize Windows 11 for Gaming

By Fernando Oliveira May 2026 12 min read Tech Guide

Windows 11 ships with a lot of background processes, visual effects, and settings that hurt gaming performance. None of these tweaks require third-party software — just built-in Windows settings. Follow all of them and you'll notice the difference.

High-Impact Tweaks

Step 1 High Impact

Set Power Plan to High Performance

Open Control Panel → Power Options and select High Performance. This prevents Windows from throttling your CPU to save power during gaming. On laptops, always plug in before gaming and set this plan.

Step 2 High Impact

Enable Hardware-Accelerated GPU Scheduling (HAGS)

Go to Settings → System → Display → Graphics → Change default graphics settings and turn on Hardware-accelerated GPU scheduling. This reduces latency between CPU and GPU, improving frame times noticeably.

Step 3 High Impact

Update Your GPU Drivers

Outdated GPU drivers are one of the most common causes of poor gaming performance and crashes. Download the latest drivers directly from NVIDIA GeForce Experience or AMD Adrenalin. Do a clean install — always select "clean installation" during the process.

Step 4 High Impact

Disable Startup Programs

Press Ctrl + Shift + Esc to open Task Manager, go to the Startup tab, and disable everything that doesn't need to run at boot. Discord, Spotify, Steam, OneDrive — all of these consume RAM and CPU before you even open a game.

Step 5 Medium Impact

Turn on Windows Game Mode

Go to Settings → Gaming → Game Mode and make sure it's On. Game Mode prioritizes CPU and GPU resources for your active game and reduces background Windows Update activity during sessions.

Step 6 Medium Impact

Disable Xbox Game Bar

Go to Settings → Gaming → Xbox Game Bar and turn it Off. Game Bar runs in the background even when you're not using it and consumes resources. If you need to record clips, use NVIDIA ShadowPlay or AMD ReLive instead.

Step 7 Medium Impact

Adjust Visual Effects for Performance

Search for Adjust the appearance and performance of Windows in the Start menu. Select Adjust for best performance or manually uncheck animations and shadows. This frees up CPU cycles that Windows was wasting on visual effects you never notice.

Step 8 Medium Impact

Set Your Monitor to Its Maximum Refresh Rate

Right-click the desktop → Display settings → Advanced display and set the refresh rate to the maximum your monitor supports. Windows sometimes defaults to 60Hz even on a 144Hz monitor. This is an easy miss that makes a huge visual difference.

Step 9 Medium Impact

Disable Background Apps

Go to Settings → Apps → Installed apps, click on any app you don't need, select Advanced options, and set Background apps permissions to Never. Do this for Microsoft Teams, Cortana, and any apps you rarely use.

💡 After All Tweaks Restart your PC after making all these changes. Some settings — especially driver updates and startup changes — only fully apply after a clean reboot. Then monitor your temperatures and FPS using MSI Afterburner to see the before/after difference.
⚠️ Don't Use "Optimization" Software Apps like CCleaner, Advanced SystemCare, or any "PC optimizer" often do more harm than good. They delete files Windows needs, break registry entries, and frequently bundle unwanted software. Everything in this guide uses only built-in Windows tools.

Common Mistakes to Avoid

Even experienced users make these errors when trying to optimize Windows 11 for gaming. Avoid them and you'll save yourself hours of troubleshooting.

  • Leaving V-Sync on in-game while also enabling it in GPU control panel — double V-Sync adds input lag. Pick one location or turn it off entirely if you're not seeing screen tearing.
  • Forgetting to update after a Windows major update — big Windows updates often reset some settings (power plan, Game Mode) back to defaults. Check these after any major Windows update.
  • Disabling Windows Defender — some guides tell you to disable antivirus entirely. Don't. The performance cost of Defender is minimal and the risk of running without any protection isn't worth it. Instead, check our best free antivirus for gamers guide for lightweight options that are specifically designed for gaming.
  • Using third-party "driver updaters" — only update GPU drivers from NVIDIA or AMD's official websites. Third-party driver tools install wrong or outdated versions regularly.
  • Enabling every tweak you read about online — some registry tweaks floating around gaming forums are outdated or outright harmful. Stick to the verified settings in this guide.

Advanced Tips for Extra Performance

Once you've done the nine core tweaks, these advanced optimizations can squeeze out additional improvements — especially for competitive gaming where every frame matters.

Advanced Tip: MSI Mode for Your GPU Message Signaled Interrupts (MSI) mode reduces GPU interrupt latency, which translates to more consistent frametimes. You can enable it through GPU driver settings or with a free tool like MSI Utility v3. This is a more advanced tweak — back up your system before attempting it.

Enable Resizable BAR (if your hardware supports it)

Resizable BAR allows your CPU to access the full GPU video memory instead of just a 256MB window. This can add 5–15% FPS in supported games. Check your motherboard BIOS for a "Resizable BAR" or "Above 4G Decoding" option. Both your CPU and GPU need to support it — most 2020+ hardware does.

Disable Nagle's Algorithm for Online Games

Nagle's Algorithm bundles small network packets together to improve efficiency — which adds latency in games. You can disable it via a registry edit: navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters\Interfaces, find your network adapter's entry, and add two DWORD values: TcpAckFrequency = 1 and TCPNoDelay = 1. This pairs well with the other fix high ping tweaks.

Use DirectX 12 Ultimate where available

Modern games that support DX12 or Vulkan generally perform better and stutter less than DX11 on Windows 11. If your game has a DX12 or Vulkan option in its settings, use it — Windows 11 is specifically optimized for the newer graphics APIs.

Schedule Windows Update for off-hours

Go to Settings → Windows Update → Advanced options → Active hours and set your gaming hours. This prevents Windows from downloading updates and potentially restarting your PC mid-session.

Frequently Asked Questions

Does Windows 11 actually perform better than Windows 10 for gaming?

In most cases, the difference is marginal — a few percent either way depending on the game and hardware. Windows 11 has better support for DirectStorage and Auto HDR, which can benefit newer games. The bigger factor is always your hardware and whether you've applied the optimizations in this guide, not which version of Windows you're running.

Will these tweaks void my warranty or damage my PC?

No. Every tweak in this guide uses built-in Windows settings — no registry edits, no third-party tools, no overclocking. They are completely safe and reversible. If you ever want to undo a change, simply reverse the setting to its default value.

How much FPS improvement should I expect?

It depends heavily on your starting point. If you've never optimized your PC before, you can expect 10–30% improvement in average FPS and noticeably smoother frametimes — especially from the power plan, GPU drivers, and startup program tweaks. If your PC was already partially optimized, gains will be smaller but still meaningful.

The Verdict

These nine tweaks take about 20 minutes total and cost nothing. The biggest wins are the power plan, HAGS, updated GPU drivers, and disabling startup programs. Do those four first — you'll notice the improvement before you even finish the list.

Windows 11 is a capable gaming OS once you cut through the bloat. The defaults aren't optimized for gaming — but with these changes, they will be.
