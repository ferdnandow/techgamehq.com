How to Reduce Input Lag on Any Monitor

May 2026 6 min read Tech

Input lag is the delay between moving your mouse or pressing a key and seeing the result on screen. Even a few milliseconds can make games feel sluggish. Here's how to minimize it — mostly for free.

What Causes Input Lag?

Input lag comes from multiple sources stacking up: your monitor's response time, V-Sync adding delay, Windows processing, and your GPU render queue. The good news is most of these can be reduced through settings alone.

Fix 1: Turn Off V-Sync

Free — Do This First

Disable V-Sync In-Game and in GPU Settings

V-Sync synchronizes your FPS to your monitor refresh rate, which sounds good but adds 1-3 frames of input lag. Turn it off in every game's settings. Also disable it in your GPU control panel (NVIDIA Control Panel or AMD Radeon Software) for a global override.

If you get screen tearing without V-Sync, use G-Sync or FreeSync instead — they eliminate tearing without the input lag penalty.

Fix 2: Enable Game Mode in Windows

Free

Windows Game Mode Reduces Background Interference

Press Windows + I → Gaming → Game Mode → Toggle ON. This prioritizes your game for system resources and reduces background process interference, which can shave off a few ms of input lag.

Fix 3: Use Your Monitor's Game Mode

Monitor Setting

Enable Game Mode or Low Input Lag Mode on Your Monitor

Most gaming monitors have a dedicated "Game Mode" or "Low Input Lag" setting in their OSD menu (the buttons on the side or bottom of your monitor). This disables image processing features that add delay. It might make the image look slightly less vibrant but the responsiveness improvement is real.

Fix 4: Set Your Monitor to Its Maximum Refresh Rate

Free — Often Missed

Check Your Refresh Rate in Windows Settings

Many people have a 144Hz monitor running at 60Hz because Windows defaulted to the lower rate. Right-click your desktop → Display Settings → Advanced Display → choose the highest refresh rate available. This alone can make games feel dramatically smoother.

💡 Use DisplayPort Not HDMI If your monitor supports both, use DisplayPort over HDMI. DisplayPort handles high refresh rates more reliably and can reduce input lag compared to some HDMI implementations. Use HDMI 2.1 only if you need it for a console.

Fix 5: Limit FPS Slightly Below Your Refresh Rate

Free — Advanced

Cap FPS at Refresh Rate Minus 3-4 Frames

If your monitor is 144Hz, cap your FPS at 141. This keeps your GPU from building up a render queue (which adds lag) while staying close to your maximum refresh rate. Use RTSS (RivaTuner Statistics Server, comes with MSI Afterburner) to set a precise FPS cap.

Fix 6: Turn Off Unnecessary Monitor Features

In your monitor's OSD menu, disable these features that add processing delay:

  • HDR — adds significant processing lag unless you have a true HDR monitor
  • Dynamic Contrast — turn off
  • Motion Blur Reduction / ULMB — can add lag on some monitors
  • Overdrive/Response Time — set to Medium, not Maximum (causes overshoot)

Quick Checklist

Do these right now and your games will feel noticeably more responsive:

  1. Turn off V-Sync in-game and GPU settings
  2. Enable Game Mode in Windows
  3. Enable Game Mode on your monitor
  4. Confirm your monitor is running at max refresh rate
  5. Switch to DisplayPort if you're using HDMI
  6. Cap FPS just below your refresh rate
