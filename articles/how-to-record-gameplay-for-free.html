How to Record Your Gameplay for Free

By TechGameHQ Staff May 2026 6 min read Tech Guide

You don't need expensive software to record high-quality gameplay. The best recording tools are completely free — some are built into your GPU drivers. Here's which one to use depending on your setup, and how to configure it properly.

Short answer: If you have an NVIDIA GPU, use ShadowPlay. AMD GPU, use ReLive. Neither? Use OBS Studio. All three are free and produce great results.

Option 1: NVIDIA ShadowPlay (GeForce Experience)

ShadowPlay uses your GPU's dedicated video encoder, meaning it records at near-zero performance cost. You won't even notice it's running.

Setup

How to set up ShadowPlay

Install GeForce Experience from NVIDIA's website. Open it, go to Settings → In-Game Overlay and enable it. Press Alt + Z in any game to open the overlay. Click Record to start/stop, or use Instant Replay to automatically save the last 5–20 minutes of gameplay. Set quality to 1080p 60fps for the best balance.

Option 2: AMD ReLive (Adrenalin Software)

AMD's equivalent to ShadowPlay. Same idea — hardware-accelerated recording with minimal performance impact.

Setup

How to set up ReLive

Install AMD Adrenalin drivers from AMD's website. Open the software, go to Gaming → Recordings and enable ReLive. Press Alt + Z to open the overlay in-game. Enable Instant GIF or full recording depending on what you need. Set bitrate to 50Mbps for 1080p 60fps quality.

Option 3: OBS Studio (Any GPU)

OBS is the most powerful free recording and streaming tool available. It works on any hardware and gives you full control over quality, format, and output. Slightly more setup than the GPU options but worth it.

Step 1

Download and install OBS

Download OBS Studio free from obsproject.com. Run the Auto-Configuration Wizard on first launch — it will detect your hardware and set optimal settings automatically.

Step 2

Add a Game Capture source

In OBS, click the + button under Sources and add Game Capture. Set it to Capture any fullscreen application. This automatically detects your game when you launch it.

Step 3

Set output settings

Go to Settings → Output and set the recording path to a drive with plenty of space. Set encoder to NVENC (NVIDIA) or AMF (AMD) for hardware encoding — this cuts CPU usage significantly. For quality: Rate Control = CQP, CQ Level = 18–23 (lower = better quality, larger files).

Step 4

Set a hotkey and start recording

Go to Settings → Hotkeys and assign a key to Start/Stop Recording (e.g., F9). Press it in-game to start recording. OBS runs in the background — you won't see it while gaming.

💡 Storage Warning Gameplay recordings eat storage fast. 1 hour of 1080p 60fps at high quality = 10–30GB depending on settings. Always record to your HDD (not your SSD) and keep an eye on free space. Delete raw recordings after editing or uploading.

Recommended Settings for YouTube Upload

  • Resolution: 1920×1080
  • Frame rate: 60fps
  • Encoder: NVENC / AMF (hardware) or x264 (software)
  • Bitrate: 50–80 Mbps for local recording
  • Format: MP4 (compatible with everything)

The Verdict

If you have an NVIDIA or AMD GPU, use the built-in recording tool first — it's the easiest setup with the lowest performance impact. If you want more control, or you're on integrated graphics, OBS Studio is the best free option available and used by professional streamers worldwide.

Recording your gameplay is one of the best ways to improve — watching your own footage reveals mistakes you miss in the moment. Start recording, start learning.
